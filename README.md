# LLM training digest

This repository publishes PDF digests on **applied fine-tuning of small LLMs**: choosing a base model and training mode, training data and validation, end-to-end pipeline stages (dataset validation → GPU deploy → training monitor → model retrieval → inference → evaluation), infrastructure, metrics, risks, and cost/time of a run.
